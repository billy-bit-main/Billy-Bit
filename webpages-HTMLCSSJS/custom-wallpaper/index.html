custom wallpaper.

A live desktop wallpaper built with HTML, CSS, and JS — runs through Lively Wallpaper, a free Windows app that lets you use web pages as wallpapers. You can swap out the background image to anything you want by replacing one line of CSS. Here's how to set it up and make it yours.

setup · lively wallpaper
01
Download Lively Wallpaper

Get it from the Lively website or search it on the Microsoft Store — both are free. Install and open it.

02
Download the wallpaper files

Grab the files from the GitHub repo. You want the whole folder — index.html, styles.css, and scripts.js. Keep them together in one folder on your computer.

03
Add it to Lively

In Lively, click the + button in the top right. Choose "Open from disk" and navigate to the index.html file you downloaded. Lively will load it as a wallpaper — click OK through any prompts.

tip
If it asks how to open the file, pick "Web page". Don't pick the "HTML" app option — it won't render correctly.
04
Set it as your wallpaper

Once it loads in Lively, double-click the preview or click "Set as wallpaper". It'll go live on your desktop. Lively runs in the background — you can close the window and it keeps going.

changing the image
01
Convert your image to base64

Base64 is just a way of turning an image file into a long text string, so you can paste it directly into your CSS instead of having to link to a file. Go to base64.guru/converter/encode/image, upload your image, and copy the output string.

heads up
The output string will be very long — that's normal. The bigger the image file, the longer the string. Don't panic.
02
Open styles.css and find the background line

Open styles.css in any text editor (Notepad works, VS Code is better). Look for the line that starts with background-image: or find the url('data:image/ part — that's where the current image is embedded.

03
Swap in your base64 string

Replace everything between the quotes with your new base64 string, keeping the data:image/ prefix. The format looks like this:

styles.css · image swap 

background-image: url('data:image/jpeg;base64,PASTE_YOUR_STRING_HERE');

background-image: url('data:image/png;base64,PASTE_YOUR_STRING_HERE');
04
Save and reload in Lively

Save the CSS file. In Lively, right-click the wallpaper preview and hit Reload — or just remove and re-add the wallpaper. Your new image will be live.

note
If the image looks stretched, check the background-size property in the CSS. cover fills the screen, contain fits it without cropping. Change it to whichever looks right for your image.
